Main Column

This column is wider because it is set to "flex: 2" which means in this case, that it is 2/4 of the parent div (minus margin and padding.)

If there were 4 divs, "flex: 2" would make it 2/5 of the parent div.

If there were 5 divs "flex: 2" would make it 2/6 of the parent div.

If this div was set to "flex: 3" it would make it 3/5 of the parent div.

Sidebar One

You can move these divs around in order from left to right with "order: 1 | 2 | 3 | etc."

Feature 1

You don't need to add a margin around these divs beacuse flexbox adds the margin for you.

Feature 2

"Justify-content: flex-start | flex-end | center | space-between | space-around | stretch;" tells how to distribute divs.

"Align-items: flex-start | flex-end | center | baseline | stretch;" says how to align them.

Notice that some of the terms are the same but they do different things in each setting.

Feature 3

"Justify: space-around" makes it so that items are evenly distributed in the line with equal space around them. Note that visually the spaces aren't equal, since all the items have equal space on both sides. The first item will have one unit of space against the container edge, but two units of space between the next item because that next item has its own spacing that applies.

"Justify: space-between" makes it so that items are distributed so that the spacing between any two items (and the space to the edges) is equal.
